Programação Imperativa (CC1003)

    Dada uma base de dados com informação sobre alunos e disciplinas de cursos de uma faculdade, pretendemos consultá-la para obter informações. As estruturas de dados que vão ser usadas pelo programa, obedecem à especificação seguinte. 
     
typedef struct aluno {
  char nome[MAXNOME], codigo[MAX];
  int disc[MAX], nd;
} ALUNO;


typedef struct disciplina {
  char nome[MAXNOME];
  int ano;
} DISCIPLINA;


int NDisc = 9;
DISCIPLINA Disc[MAXDISC] = {
      {"Programacao Imperativa",1},
      {"Calculo Infinitesimal I",1},
      {"Programacao Estruturada",1},
      {"Topologia",4},
      {"Geometria Computacional",4},
      {"Probabilidades e Estatistica",2},
      {"Teoria de Numeros",3},
      {"Analise Infinitesimal",3},
      {"Logica",3}
     };	

int NAlunos = 6;
ALUNO Alunos[MAXALUNOS] = {
    {"Mariana Matias","2001018003", {2,1,9,5,3,7}, 6},
    {"Anabela Moreira", "2000020004",{6,2,3,7,8,9}, 6},
    {"Sonia Silva", "2000020005",{5,6,1,8,9},5},
    {"Joao Maria Peixoto", "2000018001",{3,4,1,5}, 4},
    {"Joao Mario Pereira","2004010005",{8,1,2,3,4,5,6},7},
    {"Maria Reis", "2000020050",{5,6,1,7,3,9},6}
  };

    No exemplo, Programacao Imperativa é a disciplina 1 e Programacao Estruturada a disciplina 3.
    As disciplinas a que um aluno está inscrito são identificadas pelos seus números de ordem no array Disc.
    O código de um aluno é uma sequência de 10 dígitos: os quatro primeiros identificam o ano de inscrição, os três seguintes o código do curso, e os três últimos identificam o aluno.

    Exercícios

    1. Escrever uma função void procura(char palavranome[]) para imprimir o código e nome dos alunos cujo nome contém uma palavra dada por palavranome. Por exemplo, se a palavra fosse "Maria", deveria imprimir: 
      Alunos cujo nome inclui Maria

2001018003 Mariana Matias
2000018001 Joao Maria Peixoto
2000020050 Maria Reis
      Não deverá ser feita qualquer ordenação dos dados. A menos que a ordenação seja explicitamente solicitada, no output deverá ser mantida a ordem de ocorrência dos dados nas estruturas de dados.

      2. Para testar o programa, pode usar os dados que constam do ficheiro dados.h. Para isso, coloque a instrução 
      #include "dados.h"
      Na próxima aula teórica serão dadas outras formas para introdução dos dados.

    2. Obter o nome de um aluno dando o seu código (retorna 0 se não existir, e 1 caso contrário): 
        int codigo2Aluno(char codigo[],char nome[])

    3. Escrever a lista de disciplinas a que um aluno está inscrito, sendo dado o nome do aluno (e retorna o número de disciplinas a que está inscrito): 
        int disciplinas_de_Aluno(char nome[])

    4. Obter os índices (das posições em Alunos dos) dos alunos que estão inscritos a uma disciplina, dando o nome da disciplina (retorna o número de alunos nessas condições) 
        int alunos_de_disciplina(char disciplina[], int indices[])

    5. Obter o número médio de disciplinas em que os alunos dum dado curso estão inscritos. Escreva uma função que retorne esse valor, arrendondado por defeito, se necessário: 
        int numero_medio(char curso[])

    6. Obter os índices das posições (em Alunos[]) dos alunos dum dado curso que estão inscritos a mais disciplinas do que o número médio a que se encontram inscritos os alunos desse curso. Escreva uma função 
        int muitas_disciplinas(char curso[], int indices[])
       para colocar os índices procurados no vector dado por indices, e que retorne o número de alunos nessas condições.

    7. Obter o número máximo de disciplinas a que os alunos que entraram num dado curso num dado ano estão inscritos. Para determinar esse valor, escreva uma função 
        int numero_maximo(char cursoano[])
      A sequência de caracteres dada por cursoano foi obtida por concatenação do ano de entrada com o código do curso (dados), ou seja, se fosse "2006" e "018", cursoano apontaria "2006018".

    8. Obter o número de disciplinas dum dado ano a que um dado aluno está inscrito. Para determinar esse valor, escreva uma função 
       int numero_disc(int posaluno,int ano)
      Assume-se que posaluno contém o índice da posição do aluno em Alunos[], sendo uma posição válida.

    9. Verificar se um dado aluno está inscrito a disciplinas de vários anos. Escreva uma função 
       int varios_anos(int posaluno)
      para o fazer. A função retorna um valor qualquer diferente de zero em caso afirmativo, e 0 (zero) caso contrário. Assume-se que posaluno contém o índice da posição do aluno em Alunos[], sendo uma posição válida.
